Project Overview: FM Portfolio (Dark & Kinetic)

Vision & Aesthetic
A high-end, single-page developer portfolio for Federica Mariani. The aesthetic is
"Awwwards-inspired": dark, minimalist, and brutalist, focusing on high-impact motion and bold
typography.

Design System (Refined)

- Palette: Avoiding pure black/white for a more sophisticated, "ink" feel.
  - Primary Background: #0E0E0E (Deep Ink Black)
  - Secondary Background: #161616 (Soft Charcoal)
  - Primary Typography: #F2F2F2 (Off-White / Ghost White)
  - Secondary/Accent: #8A8A8A (Medium Grey)
- Typography:
  - Headings: Clash Display (Bold/Extrabold) - Brutalist Sans with high character.
  - Body: Geist or Inter (Medium/Light) - Clean, modern, and highly legible.

Technical Architecture

- Framework: Next.js 15 (App Router) + React 19.
- Styling: Tailwind CSS 4.
- Motion Engine: GSAP (@gsap/react) for kinetic animations and timeline management.
- Physics: Lenis for smooth scrolling.

Section Strategy

1.  The Hook (Hero): Kinetic typography reveal using GSAP. Focus on "Senior Full-Stack & AI
    Innovation".
2.  The Story (About): Subtle scroll-triggered text reveals.
3.  The Craft (Expertise): Interactive grid or marquee showcasing the tech stack (AI, React,
    Next.js).
4.  Selected Works: Magnetic hover interactions with large image previews.
5.  The Timeline (Experience): Minimalist vertical flow with scroll-linked animations.
6.  The Connection (Contact): Bold footer with magnetic CTA buttons.

Animation Principles

- Kinetic Energy: Elements should feel like they have weight and velocity.
- Magnetic Interaction: Cursor-aware elements that attract/repel based on proximity.
- Fluidity: Seamless transitions between sections facilitated by Lenis.
